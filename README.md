# 📊 Reconciliation Dashboard

## Установка (один раз)

### 1. Установи Python
Скачай Python 3.11+ с https://python.org/downloads
При установке поставь галочку "Add Python to PATH"

### 2. Установи зависимости
Открой командную строку (cmd) и выполни:
```
pip install streamlit pandas openpyxl numpy xlrd
```

### 3. Запуск приложения

**Windows — двойной клик на `start.bat`**

Или через командную строку:
```
cd путь/к/папке/reconciliation_app
streamlit run app.py
```

Приложение откроется в браузере на http://localhost:8501

## Как использовать каждый день

### Шаг 1 — Загрузи файлы
- **Supplier File (.xls)** — файл от поставщика (ОБЯЗАТЕЛЬНО .xls, не .csv!)
- **Partner File (.csv)** — наш экспорт Partner
- **012Talk File (.csv)** — наш экспорт 012Talk

### Шаг 2 — Выбери дату и нажми Run
- Выбери правильную дату отчёта
- Нажми **▶ Run Reconciliation**

### Шаг 3 — Проверь результаты
- **Difference = 0.00 NIS** → всё совпало ✅
- **Supplier Only** → номера только у поставщика (обычно за прошлую дату)
- **Our Only** → наши номера без поставщика (обычно поздние транзакции)
- **Pending** → проверить на следующий день

### Шаг 4 — Скачай и сохрани
- **📥 Download Excel Report** → сохрани в SharePoint папку месяца
- **💾 Save to Monthly History** → добавит строку в месячный трекер

### Шаг 5 — В конце месяца
- Перейди в **📅 Monthly Summary** в боковом меню
- Выбери месяц
- Нажми **📥 Download Monthly Report**

---

## Важные правила

⚠️ **Файл поставщика ВСЕГДА загружать в формате .xls** — если сохранить как .csv, номера телефонов обрезаются!

⚠️ **PENDING_CANCELLATION** — клиент запросил отмену, поставщик ещё не решил. Проверить на следующий день.

⚠️ **REFUND строки** — возвраты за прошлый период, приходят в конце месяца.

⚠️ **REWARD и REFUND_REWARD** — автоматически исключаются из расчётов.
